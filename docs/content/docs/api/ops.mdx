# Operations

Core mathematical operations on arrays. All operations support broadcasting and run on GPU when available.

## Arithmetic

### add

Element-wise addition.

```cpp
array add(const array& a, const array& b, StreamOrDevice s = {});
array operator+(const array& a, const array& b);
```

**Examples:**

```cpp
auto a = array({1.0f, 2.0f, 3.0f});
auto b = array({4.0f, 5.0f, 6.0f});
auto c = add(a, b);       // [5.0, 7.0, 9.0]
auto d = a + b;           // Same as add(a, b)

// Broadcasting
auto e = add(a, array(10.0f));  // [11.0, 12.0, 13.0]
```

### subtract

Element-wise subtraction.


### multiply

### divide

### power

Element-wise power.

```cpp

```

**Examples:**

```cpp

auto b = power(a, array(2.0f));  // [1.0, 4.0, 9.0]
auto c = power(array(2.0f), a);  // [2.0, 4.0, 8.0]
```

### where

Select elements based on condition.

```cpp
array where(const array& condition, const array& x, const array& y, StreamOrDevice s = {});
```

**Examples:**

```cpp
auto a = array({1, 2, 3, 4, 5});
auto cond = greater(a, array(3));
auto b = where(cond, a, zeros_like(a));  // [0, 0, 0, 4, 5]
```

## Broadcasting Rules

Operations automatically broadcast arrays:

```cpp
auto a = ones({3, 1});    // Shape: {3, 1}
auto b = ones({1, 4});    // Shape: {1, 4}
auto c = add(a, b);       // Shape: {3, 4}

auto d = ones({3, 4});
auto e = ones({4});       // Broadcasts to {1, 4} then {3, 4}
auto f = add(d, e);       // Shape: {3, 4}
```

Broadcasting rules:
1. Dimensions are compared from right to left
2. Dimensions must be equal or one of them must be 1
3. Size-1 dimensions are broadcast to match the other
